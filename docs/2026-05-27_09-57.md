# 修改记录: 2026-05-27 09:57

## 诊断
cp5100 步态已优雅自然(tracking=0.90, smoothness=2.80, dof_acc=276k)，但域随机化覆盖不足：
- disturbance/delay 仍关闭
- 摩擦/电机/PD 随机范围偏窄
- 未模拟负载变化
- push 力度偏弱

## 修改内容
| 参数 | 旧值 | 新值 | 目的 |
|------|------|------|------|
| disturbance | False | True | 随机外力扰动 |
| disturbance_range | [-2, 2] | [-3, 3] | 更大干扰力 |
| disturbance_interval | 8s | 6s | 更频繁干扰 |
| randomize_base_mass | False | True | 模拟不同负载 |
| added_mass_range | [-0.1, 0.3] | [-0.15, 0.35] | 扩大负载范围 |
| friction_range | [0.5, 1.25] | [0.3, 1.5] | 更滑/更粘地面 |
| motor_strength_range | [0.9, 1.1] | [0.85, 1.15] | 更大电机误差 |
| kp_range | [0.8, 1.2] | [0.7, 1.3] | 更大PD刚度波动 |
| kd_range | [0.8, 1.2] | [0.7, 1.3] | 更大PD阻尼波动 |
| push_interval_s | 15 | 10 | 更频繁推搡 |
| max_push_vel_xy | 0.5 | 0.8 | 更强推力 |

## 预期效果
所有域随机化项OpenDoge级强度开启。步态从"理想条件优雅"升级为"各种扰动下依然优雅"。Sim2Real差距大幅缩小。
